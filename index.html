margin合并

Posted on 2017-07-07

margin合并

一. 包含关系

<box><bar></bar></box>

.box{ width:300px; height:300px; background:#f00; margin-top:20px; }

.bar{ width:300px; height:300px; background:#eee; margin-top:50px; }

父级margin-top不会叠加，而是取最大值，合并

解决方法

  1. 给父级加border/padding
  2. 给父级加属性overflow:hidden
二.上下关系

.box1{ margin-bottom:20px; }
.box2{ margin-top:30px; }
(两个之间的距离是30，而不是50，取最大值，而不是相加)
解决方法

  1. 给父级加border/padding
  2. 给父级加属性overflow:hidden
三.左右关系

外边距左右不合并，只合并上下

四.清除浮动

如果父级浮动了，子级就不需要清除浮动了
